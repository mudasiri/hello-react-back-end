## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to have the following in local computer:

- [ ] install Ruby
- [ ] install Ruby on Rails
- [ ] install PostgreSQL
- [ ] install react
- [ ] install react dom

### Setup

Clone this repository to your desired folder:

- [ ] Open terminal
- [ ] git clone the backend: https://github.com/mudasiri/hello-react-back-end.git
- [ ] git clone the frontend: https://github.com/mudasiri/hello-react-front-end.git
- [ ] cd budget-app
  

### Install

Install this project with:

 - [ ] bundle install

Install this project client with:

 - [ ] yarn install


### Run the project

To run the project, run the following command:

- [ ] rails server

To run the webpack to build the client, run the following command:

- [ ] yarn run build
